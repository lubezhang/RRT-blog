## Mice 
一个使用 React+Redux+koa 的简单blog系统。

## 安装
### 开发阶段(http://localhost:8360)
可以实现react组件、sass不刷新自动更新
~~~javascript
npm install
npm run dev  
~~~

### 上线
~~~javascript
npm install
npm start (pm2 start pm2.json) 
~~~
